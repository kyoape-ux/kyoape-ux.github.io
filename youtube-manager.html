系統概覽

本系統用於管理多媒體組在 YouTube、Facebook、Instagram 三大平台的影片發布紀錄與數據追蹤，提供儀表板分析、報表匯出等功能。

主要功能一覽

功能說明
儀表板KPI 卡片、觀看趨勢圖、平台占比、類別分布、Top 5 排名
影片管理新增 / 編輯 / 刪除影片，三大平台數據追蹤
IG 管理獨立管理 Instagram 貼文，含觀看、互動、觸及等數據
匯出報表月報表、年度報表，可列印或儲存為 PDF
自動取得透過 YouTube API 自動抓取影片觀看數、按讚數等數據
系統資料同步至 Google Sheets，多人同時操作不會互蓋。本機模式下資料僅存在瀏覽器，重新整理仍保留。

角色與權限

系統設有三種身份，登入後可隨時點擊右上角身份按鈕切換。

身份權限適用對象
唯讀僅瀏覽儀表板、列表、報表（無法新增/編輯）主管、外部查閱
組員可新增、編輯、刪除影片及 IG 貼文多媒體組成員
組長完整權限：含系統設定、績效目標、資料管理組長
密碼由組長管理，請勿外流。忘記密碼請聯繫組長重設。

新增影片

填寫影片資訊與各平台數據，儲存後自動同步至 Google Sheets。

必填欄位

  • 影片標題 — 輸入影片名稱
  • 發布日期 — 影片上線日期

選填欄位

  • 影片類型 — Long（長片）/ Short（短片/Reels）
  • 內容類別 — 疾病衛教、特色醫療、新服務流程、醫院形象、員工活動、教學、其他
  • 影片時長 — 格式 3:25（分:秒）
  • 負責企劃 / 影片製作 — 人員姓名

平台數據

平台可填欄位
YouTube連結、觀看、按讚、留言、曝光、CTR、觀看比例、平均時長
Facebook連結、觀看、按讚/心情、留言、分享、觸及人數
Instagram連結、觀看、按讚、留言、分享、觸及人數

自動取得 YT 數據

貼上 YouTube 連結→點擊「自動取得 YT 數據」→系統自動填入
需先在「系統設定」頁面設定 YouTube Data API Key（可至 Google Cloud Console 免費申請）。

匯出報表

產生月報表與年度報表，格式適合列印或儲存為 PDF。

月報表

選擇年月→產生月報表→新視窗開啟→列印 / PDF

月報表包含：

  • 4 張 KPI 卡片（上傳部數、總觀看、達成率）
  • 三大平台觀看數與進度條
  • 觀看次數 TOP 5 影片排名
  • 完整影片清單（含各平台數據）
  • IG 貼文統計區塊

年度報表

  • 年度 KPI 總覽
  • Q1~Q4 季度摘要表
  • 12 個月逐月趨勢表
  • 年度 TOP 10 影片排名
建議使用 Chrome 瀏覽器，列印時選擇「儲存為 PDF」。

API 連線

  • Apps Script Web App URL — 後端 Google Sheets API 位址
  • YouTube Data API Key — 自動取得 YT 數據的金鑰
  • 測試連線 — 驗證 API 是否正常運作

資料管理

  • 強制同步 — 從 Google Sheets 重新拉取所有資料
  • 清除本機資料 — 清除瀏覽器暫存（Sheets 資料不受影響）
「清除本機資料」不可復原，請確認 Google Sheets 已有完整資料後再執行。

常見問題

Q：換電腦後看不到之前的資料？

資料存於瀏覽器本機，換電腦需按「同步」從 Google Sheets 拉取。請確認已設定 API URL。

Q：A 電腦新增後，B 電腦看不到？

在 B 電腦按右上角「同步」按鈕即可取得最新資料。

Q：自動取得 YT 數據無反應？

請至「系統設定 → API 連線」確認已填入 YouTube Data API Key。需在 Google Cloud Console 啟用 YouTube Data API v3。

Q：影片狀態顯示「待抓取」？

表示影片發布超過 1 個月但觀看數仍為 0。請編輯該影片並手動填入數據，或使用「自動取得 YT 數據」。

Q：報表如何儲存成 PDF？

產生報表後點擊「列印」按鈕，在列印對話框選擇「儲存為 PDF」。

Q：IG 資料上傳後消失？

若 Google Sheets 上 IG 資料表為空，同步時不會覆蓋本機。請先「上傳至 Sheets」推送至雲端。

Q：忘記密碼？

請聯繫組長。唯讀模式不需密碼，可直接選取「唯讀」進入系統瀏覽。
