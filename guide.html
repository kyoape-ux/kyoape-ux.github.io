1 系統簡介與登入

本系統用於管理多媒體組在 YouTube、Facebook、Instagram 三大平台的影片發布紀錄與數據追蹤，提供儀表板分析、報表匯出等功能。

首次使用流程

開啟網站 → 選擇身份 → 輸入密碼 → 進入系統

資料儲存方式

  • 本機快取：資料暫存於瀏覽器 localStorage，切換裝置時不會自動同步。
  • Google Sheets 雲端：設定 API 後，新增/編輯/刪除皆會自動同步至 Google Sheets。
多台電腦使用時，請先按「同步」按鈕從 Google Sheets 取得最新資料。

2 身份權限說明

系統設有三種身份，登入後可隨時點擊右上角身份按鈕切換：

身份權限適用對象
唯讀 僅瀏覽儀表板、列表、報表（無法新增 / 編輯） 主管、外部查閱
組員 可新增、編輯、刪除影片及 IG 貼文 多媒體組成員
組長 完整權限：含系統設定、績效目標修改、資料管理 組長
密碼由組長管理，請勿外流。忘記密碼請聯繫組長重設。

4 新增影片

此頁面用於新增影片紀錄，包含基本資訊與三大平台數據。

必填欄位

  • 影片標題：輸入影片名稱。
  • 發布日期：影片上線的日期。

選填欄位

  • 影片類型：Long（長片）/ Short（短片/Reels）。
  • 內容類別：疾病衛教、特色醫療、新服務流程、醫院形象、員工活動、教學、其他。
  • 影片時長：格式為 3:25（分:秒）。
  • 負責企劃 / 影片製作：填寫人員姓名。

平台數據區

分為三大平台，各自填入該平台的數據：

平台可填欄位
YouTube連結、觀看次數、按讚、留言、曝光次數、CTR、觀看比例、平均觀看時長
Facebook連結、觀看次數、按讚/心情、留言、分享、觸及人數
Instagram連結、觀看次數、按讚、留言、分享、觸及人數

自動取得 YT 數據

貼上 YouTube 連結 → 點擊「自動取得 YT 數據」 → 系統自動填入數據
此功能需在「系統設定」頁面先設定 YouTube Data API Key。API Key 可至 Google Cloud Console 免費申請。

5 影片列表

管理所有已建立的影片紀錄，支援搜尋、篩選、批次更新等功能。

影片狀態標記

狀態圖示說明
待抓取發布超過 1 個月，但觀看數 = 0（需更新數據）
統計中發布未滿 1 個月（數據持續累積中）
已定案發布超過 1 個月且已有觀看數（數據已鎖定）
統計規則：每部影片以「發布後 1 個月內」的觀看數為 KPI 基準。

搜尋與篩選

  • 關鍵字搜尋：可搜尋標題、企劃、製作人員。
  • 篩選器：依類型（Short/Long）、類別、月份篩選。

批次操作

按鈕功能
批次更新 YT 數據自動更新所有已填入 YouTube 連結的影片數據（需 API Key）
匯入 YT Studio CSV匯入從 YouTube Studio 匯出的 CSV 檔案，自動新增影片
匯出 CSV將所有影片資料匯出為 CSV 檔案（UTF-8 編碼）

編輯 / 刪除

每筆影片右側有「編輯」及「刪除」按鈕（僅組員/組長可見）。點擊編輯會開啟彈窗表單，修改後點「更新」即可。

7 匯出報表

系統可產生月報表與年度報表，格式適合列印或儲存為 PDF。

月報表

選擇年月 → 點「產生月報表」 → 新視窗開啟報表 → 列印 / 儲存 PDF

月報表包含：

  • 4 張 KPI 卡片（上傳部數、總觀看、觀看達成率、上傳達成率）
  • 三大平台觀看數與達成進度條
  • 本月觀看次數 TOP 5 影片排名
  • 完整影片清單（含各平台數據）
  • IG 貼文統計區塊（觀看、互動、分享、儲存 + 貼文列表）

年度報表

選擇年份後點「產生年度報表」，包含：

  • 全年度 KPI 總覽
  • 各季度 (Q1~Q4) 摘要表格
  • 12 個月逐月趨勢表
  • 年度 TOP 10 影片排名
列印時請選擇「儲存為 PDF」以保存電子檔；建議使用 Chrome 瀏覽器列印效果最佳。

8 系統設定（僅組長）

績效目標

設定說明預設值
月觀看目標全平台每月觀看數目標5,000
月上傳部數目標每月應上傳的影片數6
單片獎勵門檻單部影片 YT 觀看數達此數即列入獎勵2,000

API 連線

  • Apps Script Web App URL：後端 Google Sheets API 位址。部署方式請參考程式碼註解。
  • YouTube Data API Key：用於自動取得 YT 數據的金鑰，可至 Google Cloud Console 免費申請。
  • 測試連線：驗證 API 是否正常運作。

資料管理

  • 強制同步：手動從 Google Sheets 重新拉取所有資料。
  • 清除本機資料：刪除瀏覽器中暫存的所有資料（Google Sheets 資料不受影響）。
「清除本機資料」操作不可復原，請確認 Google Sheets 已有完整資料後再執行。

9 常見問題 FAQ

Q：換電腦後看不到之前的資料？

資料存於瀏覽器本機，換電腦後需按「同步」從 Google Sheets 拉取。請確認已設定 API URL。

Q：A 電腦新增資料後，B 電腦看不到？

在 B 電腦按下儀表板右上角的「同步」按鈕，即可取得最新資料。

Q：自動取得 YT 數據無反應？

請至「系統設定 → API 連線」確認已填入 YouTube Data API Key。API Key 需在 Google Cloud Console 啟用 YouTube Data API v3。

Q：影片狀態顯示「待抓取」怎麼辦？

表示該影片發布已超過 1 個月但觀看數仍為 0。請點擊「編輯」後手動填入數據，或使用「自動取得 YT 數據」功能。

Q：報表如何儲存成 PDF？

產生報表後，點擊報表頁面的「列印」按鈕，在列印對話框中選擇目的地為「儲存為 PDF」。

Q：IG 資料上傳後消失了？

如果 Google Sheets 上的 IG 資料表（ig_posts）為空，同步時不會覆蓋本機資料。請先使用「上傳至 Sheets」將資料推送至雲端。

Q：忘記密碼怎麼辦？

請聯繫組長。唯讀模式不需密碼，可直接選取「唯讀」身份進入系統瀏覽。
